✦ ACCORD DU PARTICIPE PASSÉ :
Avec avoir → pas d'accord avec le sujet : Elle est contente qu'il ait parlé.
Avec être → accord avec le sujet : Je regrette qu'elle soit partie. / Bien qu'ils soient arrivés.
Verbes pronominaux → accord avec le sujet : Il est dommage qu'elle se soit levée tard.
✦ SUBJONCTIF PRÉSENT vs PASSÉ — CONTRASTE
SUBJONCTIF PRÉSENT → ACTION FUTURE / SIMULTANÉE
Je veux qu'il parle maintenant.
I want him to speak now.
SUBJONCTIF PASSÉ → ACTION ANTÉRIEURE / ACCOMPLIE
Je veux qu'il ait parlé avant midi.
I want him to have spoken before noon.
PRÉSENT
Il est possible qu'elle vienne ce soir.
It's possible she'll come tonight.
PASSÉ
Il est possible qu'elle soit venue hier.
It's possible she came yesterday.
