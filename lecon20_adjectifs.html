✦ ADJECTIFS QUI CHANGENT DE SENS SELON LA POSITION
ANCIEN
avant → former : mon ancien professeur (my former teacher)
après → old/ancient : un bâtiment ancien (an old building)
GRAND
avant → great (fig.) : un grand homme (a great man)
après → tall/big : un homme grand (a tall man)
CHER
avant → dear/beloved : mon cher ami (my dear friend)
après → expensive : un restaurant cher (an expensive restaurant)
PROPRE
avant → own : ma propre voiture (my own car)
après → clean : une voiture propre (a clean car)
SEUL
avant → only : la seule solution (the only solution)
après → alone : une femme seule (a woman alone)
PAUVRE
avant → unfortunate : le pauvre homme (the poor fellow)
après → financially poor : un homme pauvre
