CE · CET
CETTE · CES

Choisissez le bon adjectif démonstratif selon le genre et le nombre du nom.

CE
Masc. consonne
ce livre · ce film ce café · ce bureau
CET
Masc. voyelle / h
cet homme · cet arbre cet hôtel · cet avion
CETTE
Féminin
cette fille · cette idée cette école · cette robe
CES
Pluriel m + f
ces livres · ces filles ces idées · ces hommes
📍 -CI = PROCHE (THIS HERE)
ce livre-ci   cette robe-ci   ces photos-ci
🔭 -LÀ = LOIN (THAT THERE)
ce film-là   cette voiture-là   ces gens-là
