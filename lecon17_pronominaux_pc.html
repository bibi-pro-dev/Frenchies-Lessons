⚠ RÈGLE D'ACCORD : Le PP s'accorde avec le sujet pour les verbes pronominaux réfléchis.
Exception : si le verbe a un COD, le PP ne s'accorde pas avec le sujet.
Elle s'est lavée. (accord) vs Elle s'est lavé les mains. (pas d'accord — COD = les mains)
