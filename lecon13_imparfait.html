✦ FORMATION DE L'IMPARFAIT
ÉTAPE 1 — TROUVER LE STEM
Prendre le nous du présent
→ retirer -ons
parlons → parl- / finissons → finiss-
stem + terminaison
Stem = base imparfait pour tous les verbes sauf être

✦ QUAND UTILISER L'IMPARFAIT ?
HABITUDE PASSÉE
Action régulière ou répétée dans le passé
Quand j'étais enfant, je jouais souvent dehors.
ACTION EN COURS
Action qui était en train de se dérouler
Il pleuvait quand je suis sorti.
DESCRIPTION DU PASSÉ
État, émotion, contexte passé
La maison était grande et calme.
SOUHAIT / POLITESSE
Avec vouloir, pouvoir pour atténuer
Je voulais vous demander un service.
